# Contributing to Confiaro

Thank you for your interest in Confiaro.

This repository contains **public documentation, architectural descriptions, interface specifications, and reference materials** related to the Confiaro project.

We welcome contributions that help improve clarity, accuracy, and overall quality of the public-facing materials.

---

## ‚úÖ What You Can Contribute

You are welcome to contribute in the following areas:

- Documentation improvements
- Typo fixes and language corrections
- Architectural clarifications
- Privacy and security discussions
- API specification refinements (non-operational)
- UI/UX documentation and examples
- Internationalization (i18n) texts

All contributions must remain **non-operational** and **non-sensitive**.

---

## üîí What Is Out of Scope

For security and privacy reasons, the following are **not accepted** in this repository:

- Core conversion logic
- Execution workflows
- Wallet infrastructure
- Liquidity management
- Rate engines
- Internal monitoring systems
- Production backend code

These components are intentionally kept private.

---

## üìù How to Contribute

1. Fork this repository
2. Create a new branch for your change
3. Make your changes clearly and concisely
4. Open a pull request with a brief explanation

For larger changes, please open an issue first to discuss the proposal.
